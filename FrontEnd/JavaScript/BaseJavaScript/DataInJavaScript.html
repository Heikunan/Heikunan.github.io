# JavaScript 中的数据

# JavaScript 基础数据类型

值类型

  • boolean
  • null
  • undefined
  • number
  • string
  • symbol

值类型存储的都是值，是没有函数可以调用的，但是 JS 做了特殊处理，在调用方法的时候会转化为其相符的引用类型再进行操作,

var a = 'abc';
console.log(a.indexOf('1'))
// 等价于
var a = 'abc';
// a -> var a = new String('abc')
console.log(a.indexOf('1'))

引用类型

  • object
  • function
  • array

在 JS 中，除了值类型其他的都是对象类型。对象类型和值类型不同的是，值存储的是值，对象类型存储的是地址。当你创建了一个对象类型的时候，计算机会在内存中帮我们开辟一个空间来存放值，但是我们需要找到这个空间，这个空间会拥有一个地址。

# 判断类型

typeof 对于值类型而言，除了 null 都可以显示正确的类型

typeof 1 // 'number'
typeof '1' // 'string'
typeof undefined // 'undefined'
typeof true // 'boolean'
typeof Symbol() // 'symbol'

typeof 对于对象类型来说，除了函数都会显示 object，所以说 typeof 并不能准确判断变量到底是什么类型

typeof [] // 'object'
typeof {} // 'object'
typeof console.log // 'function'

想要判断一个对象类型的正确类型可以使用 instanceof

const Person = function() {}
const p1 = new Person()
p1 instanceof Person // true

var str = 'hello world'
str instanceof String // false

var str1 = new String('hello world')
str1 instanceof String // true

使用 Object.toString()方法可以判断正确的类型信息

Object.prototype.toString.call('') ;   // [object String]
Object.prototype.toString.call(1) ;    // [object Number]
Object.prototype.toString.call(true) ; // [object Boolean]
Object.prototype.toString.call(Symbol()); //[object Symbol]
Object.prototype.toString.call(undefined) ; // [object Undefined]
Object.prototype.toString.call(null) ; // [object Null]
Object.prototype.toString.call(new Function()) ; // [object Function]
Object.prototype.toString.call(new Date()) ; // [object Date]
Object.prototype.toString.call([]) ; // [object Array]
Object.prototype.toString.call(new RegExp()) ; // [object RegExp]
Object.prototype.toString.call(new Error()) ; // [object Error]
Object.prototype.toString.call(document) ; // [object HTMLDocument]
Object.prototype.toString.call(window) ; //[object global] window 是全局对象 global 的引用
